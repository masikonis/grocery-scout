# Grocery Scout

This application helps families shop smarter. It aggregates deals from different grocery stores and lets users create a shopping list for their fast grocery run.

[Click here to visit](https://grocery-scout.vercel.app/).

## Stack

* [Next.js](https://nextjs.org/)
* [TypeScript](https://www.typescriptlang.org/)
* [Tailwind CSS](https://tailwindcss.com/)
* [Radix UI](https://www.radix-ui.com/)
* [Vercel](https://vercel.com/)
